# beezdemand Color Palette

beezdemand Color Palette

## Usage

``` r
palette_beezdemand()
```

## Value

Named vector of brand colors.

## Examples

``` r
palette_beezdemand()
#>   primary secondary    accent      cat4      cat5      cat6      dark     light 
#> "#2B4560" "#A25F5F" "#5D8AA8" "#7D9C7F" "#8A7BA3" "#B08C6A" "#1E3242" "#F2F2F2" 
```
