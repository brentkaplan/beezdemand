# Numerical Pmax via Optimization

Numerical Pmax via Optimization

## Usage

``` r
.pmax_numerical(expenditure_fn, price_range)
```

## Arguments

- expenditure_fn:

  Function E(p) returning expenditure at price p

- price_range:

  Numeric vector c(min, max) for search interval

## Value

List with pmax, method, boundary info, and notes
