# Replace Zeros

Replaces 0 values

## Usage

``` r
ReplaceZeros(dat, nrepl = 1, replnum = 0.01)
```

## Arguments

- dat:

  Dataframe (long form)

- nrepl:

  Number of zeros to replace with replacement value (replnum). Can
  accept either a number or "all" if all zeros should be replaced.
  Default is to replace the first zero only.

- replnum:

  Value to replace zeros. Default is .01

## Value

Dataframe (long form)

## Details

Replaces specified number of 0s with replacement value.
