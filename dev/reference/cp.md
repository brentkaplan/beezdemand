# Example cross‐price dataset

A small illustrative dataset of price (x) and consumption (y) target
(target), and group (group).

## Usage

``` r
cp
```

## Format

A data frame with N rows and columns:

- id:

  unique identifier

- x:

  price of the alternative

- y:

  consumption/demand

- target:

  target (e.g., alone, own, alt)

- group:

  e.g., drug or product
