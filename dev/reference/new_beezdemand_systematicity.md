# Create a beezdemand_systematicity Object

Create a beezdemand_systematicity Object

## Usage

``` r
new_beezdemand_systematicity(results, type, call, by_var = NULL)
```

## Arguments

- results:

  Tibble with standardized columns

- type:

  Character: "demand" or "cp"

- call:

  Original function call

- by_var:

  Character vector of grouping column names (NULL if ungrouped)
