# Check Column Names

Checks to ensure column names are specified

## Usage

``` r
CheckCols(dat, xcol, ycol, idcol, groupcol = NULL)
```

## Arguments

- dat:

  Dataframe

- xcol:

  Name of x column

- ycol:

  Name of y column

- idcol:

  Name of id column

- groupcol:

  Name of group column

## Value

Dataframe

## Details

Check column names
