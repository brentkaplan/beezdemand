# Glance Method for beezdemand_fixed_grouped

Returns one row per group with model-level summaries.

## Usage

``` r
# S3 method for class 'beezdemand_fixed_grouped'
glance(x, ...)
```

## Arguments

- x:

  A beezdemand_fixed_grouped object

- ...:

  Additional arguments passed to
  [`glance.beezdemand_fixed()`](https://brentkaplan.github.io/beezdemand/reference/glance.beezdemand_fixed.md)

## Value

A tibble with group columns prepended
